mini.ComboBox

下拉列表输入框。

Extend

PopupEdit

Usage

<input class="mini-combobox" style="width:150px;" textField="text" valueField="id" 
    url="../data/countrys.txt" value="cn" showNullItem="true" allowInput="true"/>
