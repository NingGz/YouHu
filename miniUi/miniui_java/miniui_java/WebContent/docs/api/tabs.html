mini.Tabs

选项卡。

Extend

mini.Control

Usage

<div class="mini-tabs" activeIndex="0"  style="width:350px;height:200px;">
    <div title="Tab1">
        1
    </div>
    <div title="Tab2" iconCls="icon-cut" >
        2
    </div>
    <div title="Tab3" showCloseButton="true" >
        3
    </div>
    <div title="Tab4" showCloseButton="true" enabled="false">
        4
    </div>
</div>
