Description

实现响应式CSS网格布局。

  • 网格布局样式，如：.mini-col-4, .mini-col-sm-4, .mini-col-md-4, .mini-col-lg-4，一行12格数字。
  • 显示/隐藏样式，如：.mini-visible-sm, .mini-hidden-md
  • 设备尺寸：xs, sm, md, lg
  • 支持IE6+、FireFox、Chrome、Safari、Opera等。
